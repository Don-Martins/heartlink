# **App Name**: HeartLink

## Core Features:

- Predictive Vital Analysis Tool: Uses specialized logic to analyze fetal heartbeat patterns and maternity data to flag potential health risks and simulate medical insights.
- Dynamic Fetal Heartbeat Charts: Interactive data visualizations of simulated heart rate readings, featuring glassmorphism and real-time animation waves.
- Integrated OPay Demo Wallet: A luxury mockup of the OPay payment gateway, allowing mothers to simulate funding for emergencies and medicine payments.
- Automated Alert Management System: A notification framework that coordinates status changes between mothers, hospitals, and healthcare workers during emergencies.
- Role-Based Global Dashboard: Customized frontend interfaces for Mothers, Healthcare Workers, Hospitals, and Admins to manage critical maternal data.
- Medicine Dispatch & Tracking Portal: A streamlined interface for requesting medical supplies and tracking their delivery status from pharmacy to patient.
- Centralized Transaction & Health Log: A persistent simulation layer to store and manage dashboard statistics, history reports, and platform registrations.

## Style Guidelines:

- Primary Color: Deep Jade Green (#2D866A), evoking life, professional stability, and high-end healthcare.
- Background Color: Pearl Mint White (#FAFCFB), a clean, slightly tinted base that avoids harsh hospital clinical vibes.
- Accent Color: Verdant Lime (#66CC66), a high-contrast hue used for success indicators, primary action buttons, and digital wellness tracking.
- Headline and short text: 'Space Grotesk' (sans-serif) for a modern, tech-forward scientific look. Body text: 'Inter' (sans-serif) for machine-machined readability across all dashboard data.
- Refined linear icons with soft, frosted-glass background circles to match the premium healthcare branding.
- Clean, modular grid systems utilizing glassmorphic cards and wide spacing for a tranquil user experience.
- Organic heartbeat wave animations in the hero section and subtle pulse transitions when high-risk alerts occur.
